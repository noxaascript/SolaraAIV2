THE SOLARAAIV2

About
SolaraAIV2 is a solara-ai from terminal, Before I made this, I was really bored so I made this solaraai, even though it's still beta, I'm still trying

How to install?
For linux/macos/termux, you need to update & upgrade first by
pkg update && pkg upgrade -y
pkg install python
pkg install python-pip
pip install flask requests beautifulsoup4
pip install python-dotenv
chmod +x start.sh
bash start.sh
