## ✨ Overview

**SolaraAINEW** is an experimental AI system designed as a modular AI operating environment.

It combines:
- 🧠 Multi-model AI routing (Qwen / Kimi / others)
- 🌐 Browser automation system (BrowserOS)
- ⚙️ Dev tool execution layer
- 💬 Terminal-based UI system
- 🧬 Memory-driven learning system

---

## ⚡ Key Features

### 🧠 AI Brain System
- Multi-brain routing (chat / coder / reasoning / planner)
- Hybrid model selection engine
- Context memory system
- Extensible AI architecture

### 🌐 BrowserOS
- Open and fetch web pages
- Extract readable content
- AI-powered summarization
- Navigation + session tracking

### 🖥️ UI System
- Terminal OS-style interface
- Dashboard menu system
- Live typing AI response
- Loading animations

### ⚙️ Dev Engine
- Tool-based execution system
- Extensible command router
- AI-assisted automation layer
